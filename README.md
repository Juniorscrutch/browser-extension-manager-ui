# Frontend Mentor - Browser extensions manager UI solution

## Overview

This project is a solution to the Frontend Mentor challenge. It allows users to manage browser extensions with interactive features and a clean UI.

## Features

- Toggle extensions between active and inactive
- Filter extensions (All / Active / Inactive)
- Remove extensions
- Dark mode toggle
- Responsive layout
